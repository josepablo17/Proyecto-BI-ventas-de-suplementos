# Proyecto-BI-ventas-de-suplementos
Proyecto final para el curso de BI

El objetivo del proyecto es crear un dashboard que permita identificar cuales son los suplementos más rentables, separandolos por categorías, plataformas en los que se venden, ingresos por país y demás.
El dashboard contiene 6 páginas de las cuales, cada una tiene su respectivo objetivo.

Los datos se obtuvieron de la plataforma kaggle.
Se utilizo Power BI para todo el proceso ETL.
Se utilizo DAX para la creación columnas calculadas.
